# Contributing to Primality

```
$ npm install -g grunt-cli
$ git clone https://github.com/KenanY/primality.git
$ cd primality
$ npm install
```

## Building

Generates docs, minifies, and tests:

```
$ grunt
```

## Testing

Using PhantomJS:

```
$ grunt test
```

Using web browsers:

```
$ testem
```
