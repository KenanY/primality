primality(input)

# Ⓢ Ⓣ

Creates a new primality instance.

Arguments

  1. input (Mixed): A number, string, or array to check the primality of.

Returns

(Boolean): Returns true if input is prime.

Example

primality(7);
// => true

primality('13');
// => true

primality([17, 19, 23]);
// => true
